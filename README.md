
  ## Tech stack:

- Ruby "3.2.2"
- Ruby on rails "7.0.8"
- Html, css, erb
- PostreSql

### Clone repository:
```bash
git clone https://github.com/SUppL7/railtweet
```

```bash
cd railtweet
```

### Install gems:
```bash
bundle install
```

### Initialize the database

```bash
rails db:create db:migrate db:seed
```

## Serve

```bash
rails s
```
